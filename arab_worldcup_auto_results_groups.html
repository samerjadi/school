📅 المباريات والنتائج

الاحتمالات تقديرية تعليمية وليست توقعاً رسمياً؛ ويمكن ربط الصفحة بمصدر نتائج JSON لتحديث النتيجة تلقائياً بعد نهاية المباراة.

⚙️ إعداد الربط التلقائي للنتائج

الأفضل عند النشر على GitHub Pages أن يكون الرابط ملف JSON عام يتم تحديثه من GitHub Actions أو Google Apps Script؛ لأن أغلب خدمات النتائج المباشرة تحتاج مفتاح API لا يُفضّل وضعه داخل صفحة HTML عامة.

الصيغة المقبولة: [{"home":"قطر","away":"سويسرا","homeScore":1,"awayScore":0,"status":"FT"}]، أو كائن يحتوي matches / results. الحالات المقبولة للانتهاء: FT أو FINISHED أو Final أو انتهت.

ملاحظة أمنية: لا تضع مفتاح API مدفوع داخل ملف HTML منشور، بل استخدم وسيطًا مثل GitHub Actions أو Apps Script لإنتاج ملف JSON عام.
